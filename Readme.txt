./SonicTyping:
	網站首頁
		教師/學生由此登入
		教師可註冊
		學生登入時，上方教師帳號會同時更動網址，可以複製給學生使用
		
		忘記密碼目前無用
	大廳
		登入後頁面，左方可選擇功能
	課程管理/班級管理
		教師專用，右上角可加入新資料
		滑鼠移至可編輯欄位時，會呈現亮黃色，點一下進行編輯，Enter存檔，欄位內容為空時維持原樣，可利用表格上下方全部儲存鈕儲存所有編輯中欄位
	互動紀錄
		教師可看到所有自己的學生互動紀錄
		學生僅能看到自己的互動紀錄
	匯入檔案
		加入檔案可以使用.txt檔來存取。
		加入班級以及學生名單，請看底下例：
		
		"class","班級名稱"
		"student","該班級某位學生帳號","學生密碼","學生座號","學生名稱"
		(第二筆學生資料......)
		
		先填入班級資料，後面填入學生資料會自動加入該班級，
		若要加入學生至已存在的班級內請在前面使用classREF去指定其班級編號：
		
		"classREF","班級編號"
		(學生資料......)
		
		若要加入課程與單字，請看底下例：
		
		"lesson","課程名稱"
		"word","單字","中文解釋"
		(第二筆單字資料......)
		
		先填入課程資料，後面填入單字資料會自動加入該課程，
		若要加入單字至已存在的課程內請在前面使用lessonREF去指定其課程編號：
		
		"lessonREF","課程編號"
		(單字資料......)
